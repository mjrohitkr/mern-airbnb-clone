# 🏡 Wanderlust - Property Listing Web App

Wanderlust is a full-stack web application inspired by Airbnb, allowing users to explore and create property listings. The project demonstrates backend development using Node.js, Express, and MongoDB with server-side rendering using EJS.

---

## 🚀 Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Frontend:** EJS, HTML, CSS, JavaScript  
- **Architecture:** MVC Pattern  

---

## ✨ Key Features

- Browse all property listings  
- Add new listings with title, description, image, price, and location  
- Server-side rendering using EJS templates  
- RESTful routing structure  
- Clean and modular code organization  

---

## 📁 Project Structure
models/ → Database schemas
views/ → EJS templates
routes/ → Application routes
public/ → Static assets (CSS, JS)
utils/ → Error handling utilities
app.js → Main server file
